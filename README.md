# Generative AI Exercises – Task Manager

This repository contains the Python implementation of a **Task Manager** project along with analysis notes for each file. The exercises were completed as part of a generative AI module focused on understanding and analyzing existing codebases.

---

## Project Structure
generative-ai-exercises/
│
├─ cli.py                # Command-line interface for managing tasks
├─ models.py             # Task, TaskPriority, and TaskStatus classes
├─ storage.py            # TaskStorage class for saving/loading tasks
├─ task_manager.py       # Main TaskManager logic
├─ task_list_merge.py    # Task list merging and conflict resolution
├─ task_parser.py        # Parse tasks from free-form text
├─ task_priority.py      # Priority-related utilities
│
├─ cli_analysis.md       # Notes and analysis for cli.py
├─ models_analysis.md    # Notes and analysis for models.py
├─ storage_analysis.md   # Notes and analysis for storage.py
├─ task_manager_analysis.md  # Notes and analysis for task_manager.py
├─ task_list_merge_analysis.md # Notes and analysis for task_list_merge.py
├─ task_parser_analysis.md # Notes and analysis for task_parser.py
├─ task_priority_analysis.md # Notes and analysis for task_priority.py
│
└─ README.md             # This file
